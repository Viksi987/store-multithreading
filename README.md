# Магазин – Многонишково Java приложение

Конзолно Java приложение, което симулира продажба на стоки в магазин с използване на многонишково програмиране.

##  Описание
Приложението моделира магазин, в който няколко купувача пазаруват едновременно.  
Всеки купувач има кошница със стоки и покупката му се изпълнява като отделна нишка.  
Достъпът до наличните количества е синхронизиран, за да се предотвратят грешки при конкурентна работа.

##  Функционалности
- Съхраняване на налични количества от стоки
- Продажба на стоки от кошници на купувачи
- Конкурентно изпълнение на минимум 10 купувача
- Използване на фиксиран пул от 4 нишки
- Синхронизиран достъп до споделени ресурси
- Показване на крайните наличности след всички покупки

##  Многонишковост
- Всеки купувач е реализиран като `Runnable`
- Използва се `ExecutorService` с `FixedThreadPool(4)`
- Методът за продажба е `synchronized`, за да се избегне race condition

##  Използвани технологии
- Java
- Maven
- ExecutorService
- Multithreading

## Стартиране
Стартира се класът `Main`, който:
1. Създава магазина
2. Стартира задачите за купувачите
3. Изчаква всички нишки да приключат
4. Извежда финалните наличности в магазина
